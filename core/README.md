# Lounge Asia
 
Lounge Asia - это backend-система для цифровизации ресторана, включающая управление меню, товарами, заказами и пользовательскими взаимодействиями через REST API. Архитектура проекта ориентирована на масштабирование и дальнейшую интеграцию с платёжными сервисами, доставкой и фронтенд-клиентом.

**Статус проекта:** в разработке  
**Покрытие тестами:** в процессе развития

## Технологии
- [Python](https://www.python.org/)
- [Django](https://www.typescriptlang.org/)
- [Django REST Framework](https://www.django-rest-framework.org/)
- [PostgreSQL](https://www.postgresql.org/)

## Функциональность

- управление меню и категориями
- работа с товарами ресторана
- создание и обработка заказов
- REST API для взаимодействия с frontend-клиентом
- аутентификация пользователей через JWT
- фильтрация данных через API
- подготовка к дальнейшему подключению оплаты и доставки

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ArbeKenn/lounge-asia.git
cd lounge-asia
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

### 3. Активация виртуального окружения
#### Windows
```bash
venv\Scripts\activate
```
#### Linux / macOS
```bash
source venv/bin/activate
```

### 4. Установка зависимостей
```bash
pip install -r requirements.txt
```

## Тестирование
